Помощь

Инструкция по использованию

1. Загрузка оценок: Для начала работы загрузите файл с оценками учеников. В нем должны быть следующие столбцы: имя ученика, класс и оценки по предметам (Информатика, Физика, Математика, Литература, Музыка).

2. Создание/Редактирование таблицы: Вкладка "Создание/Редактирование" позволяет редактировать данные учеников. Вы можете изменять оценки прямо в таблице и добавлять новых учеников.

3. Статистика (таблица): Вкладка "Статистика (таблица)" отображает статистику по оценкам учеников по каждому классу. В таблице отображаются средние значения и медианы по каждому предмету.

4. Статистика (графики): Вкладка "Статистика (графики)" строит графики, показывающие распределение оценок по предметам для всех классов. Графики обновляются автоматически при изменении данных.

5. Добавление нового ученика: Для добавления нового ученика заполните форму, указав ФИО, класс и оценки по каждому предмету.

6. Удаление ученика: Для удаления ученика нажмите на кнопку "Удалить" рядом с его данными в таблице.

Типы графиков

Графики показывают распределение оценок (1-5) по каждому предмету в процентах. Это позволяет наглядно увидеть, как оцениваются ученики по каждому предмету в различных классах.

О программе

Эта программа предназначена для учителей и администраторов учебных заведений, которые хотят автоматизировать процесс ведения журнала оценок. Программа позволяет загружать данные, редактировать оценки, просматривать статистику по классам и предметам, а также строить графики.
